# 📖 ShortStories — Литературная утилита

Приложение на Java Swing для удобного чтения кратких пересказов литературных произведений.

---

## 🎯 Функционал

- 📚 **Краткие пересказы** произведений русской и зарубежной классики
- 🏷️ **Категории:** Романы, Повести, Стихотворения
- 📝 **Биографии писателей** (в разработке)
- 🖥️ **Простой и интуитивный интерфейс** с вкладками
